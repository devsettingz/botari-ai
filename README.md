# Botari AI Monorepo

This repository contains both the backend and frontend for Botari.

## Structure
- **botari-api** → Express + Neon Postgres backend
- **botari-frontend** → React + Vite frontend

## Features
- Business onboarding
- JWT authentication
- Real-time dashboard analytics
- Subscription management

## Deployment
- Backend → Render
- Frontend → Vercel
